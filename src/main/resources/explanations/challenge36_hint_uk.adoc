Цю задачу можна вирішити, отримавши рядки, що зберігаються в регістрах. Ви можете використовувати будь-який інструмент налагодження. Ми покажемо, як вирішити за допомогою GDB та Radare2.

З GDB:
1. `gdb -q wrongsecrets-advanced-c-windows.exe` для відкриття GDB та завантаження виконуваного файлу
2. `disas main` для дизасемблювання функції `main`
3. `b execute` для встановлення точки зупинки у функції `execute`
4. `set args 1234567812345678` для встановлення аргументів програми
5. `r` для запуску програми
