Приватний ключ знаходиться у частині .ssh контейнера і був випадково зафіксований у git.

Ви можете знайти ключ:

1. Отримавши його з git у файлі [`.ssh/wrongsecrets.keys`]():
- Клонуйте репозиторій за допомогою `git clone https://github.com/OWASP/wrongsecrets`.
- Перейдіть до кореня проекту, відкрийте папку `.ssh` та знайдіть файл `wrongsecrets.keys`.
