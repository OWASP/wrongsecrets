Ви можете знайти правильний ввід для методу guess:

1. Порівнюючи хеші:
- Знайдіть контракт в огляднику https://etherscan.io/[Etherscan].
- Витягніть хеш зі сховища контракту.
- Переглядайте транзакції, відкривайте вкладку введення та декодуйте їх як UTF-8.
- Порівняйте хеші введень з транзакцій із збереженим значенням хешу.
