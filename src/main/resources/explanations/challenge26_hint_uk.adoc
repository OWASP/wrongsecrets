Ви можете знайти попередній стан одним із двох альтернативних рішень:

1. Знайдіть попередній блок через запит до https://infura.io/[Infura]:
- Знайдіть номер блоку для будь-якого блоку після створення контракту і до оновлення.
- Створіть ключ Infura на https://infura.io/[Infura].
- Шукайте стан сховища контракту для цього блоку.

2. Перегляньте створення контракту на https://etherscan.io/[Etherscan]:
- Перегляньте транзакцію створення контракту на Etherscan
- Перейдіть до state і перегляньте storage
