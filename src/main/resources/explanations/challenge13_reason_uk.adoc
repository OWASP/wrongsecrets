*Чому вам потрібно бути обережними з CI/CD та секретами*

При використанні інструментів CI/CD вам часто потрібно, щоб інструмент (наприклад, runner/агент, що виконує завдання) використовував секрет для аутентифікації у вторинному сервісі. Ексфільтрація цих секретів може стати досить цікавою.

Переконайтеся, що секрети в CI/CD правильно налаштовані та обмежені відповідними гілками та pull requests.
