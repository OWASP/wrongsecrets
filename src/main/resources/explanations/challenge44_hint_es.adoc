Este desafío se puede resolver con los siguientes pasos:

1. Encuentra el secreto con la línea de comandos
- usa `kubectl exec vault-0 -n vault -- vault kv metadata get -mount=secret wrongsecret` y echa un vistazo a los metadatos: ¿ves un mapa con un `secret`? Ese es el valor que necesitas ingresar.

2. Encuentra el secreto en Vault usando el token root:
- Cuando configuras el entorno K8s, el script te dice el valor del token root.
