*Warum es eine schlechte Idee ist, Geheimnisse in Containern zu speichern*

Wie du jetzt sehen kannst, kannst du jedes in einem Container gespeicherte Geheimnis leicht erkennen. Ob es sich um eine ENV, eine Datei oder eine andere Eigenschaft handelt: Wenn ein System sie lesen kann, kann es auch ein Mensch.
