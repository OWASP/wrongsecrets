Ви можете вирішити цей виклик такими кроками:

1. Знайдіть секрет у Vault, використовуючи записаний root-токен:
 - При налаштуванні середовища K8s скрипт повідомляє вам значення root-токена.
 - Використайте `vault login <токен>`, а потім `vault kv get secret/wrongsecret`.

2. Перегляньте конфігурацію на предмет помилок:
 - Перевірте конфігураційні файли Vault у репозиторії Git на наявність жорстко закодованих токенів.
