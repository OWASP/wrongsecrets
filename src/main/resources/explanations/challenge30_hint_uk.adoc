Є два способи вирішити цю задачу:

1. Переглядаючи сховище:
- Перейдіть до "localhost:8080" у своєму веб-браузері.
- Відкрийте консоль розробника, натиснувши inspect.
- Перейдіть до розділу Applications консолі розробника.
- Значення, пов'язане з ключем "secret", є вашою відповіддю!
2. Переглядаючи мережевий трафік:
- Перейдіть до WrongSecrets у своєму веб-браузері.
- Відкрийте консоль розробника.
- Перевірте вкладку мережі: чи можете ви помітити додатковий виклик для цієї задачі та його значення, що повертається?
