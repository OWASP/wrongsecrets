Por varias razones, "esconder" secretos en una página web no es una buena idea, incluso si es solo para uso interno. En primer lugar, los sitios web son a menudo accesibles para una variedad de partes, incluidos empleados, contratistas y proveedores. Mantener secretos en un sitio web aumenta el riesgo de que partes no autorizadas obtengan acceso a ellos a través de cualquier vector. La posibilidad de que un hacker se pivote después de cualquier compromiso aumenta considerablemente.
