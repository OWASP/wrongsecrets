1. Використовуйте браузер для отримання секрету:
- Спочатку перейдіть до https://github.com/OWASP/wrongsecrets/blob/master/.github/workflows/dast-zap-test.yml[Github Workflow]
- Знайдіть змінну середовища, за допомогою якої ми налаштовуємо базову аутентифікацію для ZAP (`ZAP_AUTH_HEADER_VALUE`)
- Декодуйте значення заголовка, закодоване в base64
- Перейдіть до `/authenticated/challenge37` та введіть ім'я користувача та пароль, отримані з попереднього кроку.
