=== Підказка для завдання 58

Це завдання демонструє дуже поширений антипаттерн безпеки: **рядки підключення до бази даних з вбудованими обліковими даними, що витікають через повідомлення про помилки**.

**Де шукати:**

1. **Спробуйте кінцеву точку помилки:** Відвідайте `/error-demo/database-connection`, щоб викликати збій підключення до бази даних
2. **Перевірте відповідь:** Програма спробує підключитися до бази даних і зазнає невдачі, розкривши рядок підключення
