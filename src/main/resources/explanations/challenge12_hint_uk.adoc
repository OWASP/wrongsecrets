Ви можете вирішити цей виклик, виконавши команду в контейнері:

1. Перегляньте Dockerfile та побачте, які операції COPY відбуваються.
2. Запустіть контейнер локально за допомогою `docker run jeroenwillemsen/wrongsecrets:<ТЕГ>`.
3. Виконайте `docker exec -it <containerID> /bin/sh` і перегляньте скопійовані файли.
