=== Підказка для завдання 60

Це завдання демонструє, як незахищений MCP-сервер (Протокол контексту моделі) може витікати секрети, що зберігаються у змінних середовища.

**Де шукати:**

1. **Окремий MCP-сервер працює на порту 8090** — відмінний від порту основної програми (8080)
2. **MCP-сервер реалізує протокол JSON-RPC 2.0**, як визначено специфікацією MCP
