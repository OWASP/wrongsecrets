Pour plusieurs raisons, "cacher" des secrets sur une page web n'est pas une bonne idée, même si c'est uniquement pour un usage interne. Avant tout, les sites web sont souvent accessibles à une variété de parties, y compris les employés, les sous-traitants et les fournisseurs. Garder des secrets sur un site web augmente le risque que des parties non autorisées y aient accès par n'importe quel vecteur. La possibilité pour un hacker de pivoter après tout compromis est considérablement augmentée.
