=== Знайдіть URL Webhook Slack

Ласкаво просимо до цього завдання, яке демонструє вразливість хардкодованих URL-адрес webhook Slack у змінних середовища!

Це завдання імітує реальний сценарій, де:

1. **URL-адреси webhook Slack зберігаються як змінні середовища** для сповіщень програми
2. **URL-адреси обфусковані**, щоб уникнути виявлення інструментами сканування секретів
3. **Ризик ротації персоналу**: Коли співробітник іде, webhook може не ротуватися, дозволяючи постійний доступ
