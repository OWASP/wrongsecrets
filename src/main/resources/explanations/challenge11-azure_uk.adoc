=== Azure Key Vault частина 3

Ми тепер використали Parameter Store безпосередньо з програми, але є проблема IAM...

Прийміть роль `cant-read-secrets` і спробуйте ескалацію привілеїв IAM. Не обманюйте, використовуючи власні ролі :)

Тут такі інструменти, як https://github.com/nccgroup/ScoutSuite[ScoutSuite], можуть дійсно допомогти виявити проблеми з вашою конфігурацією IAM.
