=== Пароль на основі Docker ENV

Чи знаєте ви, що в контейнерах Docker можна також використовувати ENV для встановлення пароля? Яка чудова ідея ділитися ним з усіма!

Ви можете легко виявити секрет, переглянувши, як були побудовані шари, або виявивши його за допомогою інструменту, наприклад https://github.com/goodwithtech/dockle[*Dockle*].
