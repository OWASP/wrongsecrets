=== Злам AES-шифрування зі слабким MD5-ключем

Уявіть, що ви аналітик безпеки, який досліджує мобільний застосунок, що обробляє конфіденційну інформацію. Ви виявляєте, що розробник використовує AES-шифрування для захисту секрету, але замість надійної Функції Деривації Ключа (KDF) він покладається на небезпечний алгоритм MD5 для виведення ключів шифрування з простого числового PIN.

Ви отримали зашифрований рядок: `k800mdwu8vlQoqeAgRMHDQ==`. Ключ для AES-шифрування виводиться шляхом взяття MD5-хешу PIN, який є числом від 0 до 99999. Ваше завдання — знайти правильний PIN.
