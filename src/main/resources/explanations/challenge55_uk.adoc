=== .ssh Бастіон?

Ми бачили багато так званих "бастіонних контейнерів". Тут люди створюють контейнер Docker і вбудовують "секрет" для автентифікації на хості всередині контейнера. Цим секретом часто є пара SSH-ключів, що використовується для автентифікації на хості.

Щоб бастіонний контейнер працював, необхідно вбудувати файл конфігурації SSH (`.ssh/config`) для спрощення налаштування SSH-з'єднання шляхом зберігання конфігурацій для конкретних хостів.

Чи можете ви знайти приватний ключ? Вставте його вміст у поле відповіді нижче.
