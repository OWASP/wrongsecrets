Найпростіший спосіб зламати PIN у цьому сценарії — виконати атаку грубою силою через обмежений діапазон можливих значень (0 до 99 999).

- Переберіть усі можливі PIN (від 0 до 99 999).
- Для кожного PIN обчисліть його MD5-хеш, щоб отримати ключ розшифрування, і спробуйте розшифрувати наданий шифртекст.
- Якщо розшифрований текст дорівнює `the answer`, ви знайшли правильний PIN.
