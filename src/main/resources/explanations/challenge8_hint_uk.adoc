Ви можете вирішити цей виклик такими кроками:

1. Отримайте секрет із журналів:
 - Використовуєте Docker-контейнер? Використайте `docker logs <containerID>`, щоб отримати журнали та знайти значення для challenge8.
 - Використовуєте K8s? Знайдіть Pod (`kubectl get pods | grep secret`), а потім `kubectl logs -f <ім'яПода>` для журналів.
