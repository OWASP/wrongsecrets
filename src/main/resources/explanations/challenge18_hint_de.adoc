1. Für den ersten Hash (MD5):
- Verwende ein Tool wie Hashcat:
- Installiere https://hashcat.net/hashcat/[Hashcat]

- Führe Hashcat auf dem Hash aus `hashcat -m 0 "2ab96390c7dbe3439de74d0c9b0b1767" /pfad/zur/datei/rockyou.txt`
