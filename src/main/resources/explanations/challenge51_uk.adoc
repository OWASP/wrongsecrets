=== Задача з відкритими секретами Docker

У цьому завданні ви вивчите важливість безпечного управління конфіденційною інформацією за допомогою секретів Docker у https://raw.githubusercontent.com/OWASP/wrongsecrets/refs/heads/master/src/main/resources/challenges/challenge-51/challenge51docker-compose.yml[файлі Docker Compose]. Секрети Docker призначені для безпечної передачі та зберігання конфіденційних даних, таких як паролі, ключі API та сертифікати, у сервісах Docker.

*Acme Inc.*, платформа електронної комерції, що швидко зростає, нещодавно зіткнулася з підозрілою активністю, що вказує на можливе компрометування конфіденційних даних клієнтів. Внутрішній аудит виявляє, що розробник випадково розкрив облікові дані бази даних.

Визначте неправильні конфігурації та повідомте пароль бази даних у полі нижче.
