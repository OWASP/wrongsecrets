Aus verschiedenen Gründen ist das "Verstecken" von Geheimnissen auf einer Webseite keine gute Idee, auch wenn es nur für den internen Gebrauch ist. In erster Linie sind Websites oft für eine Vielzahl von Parteien zugänglich, darunter Mitarbeiter, Auftragnehmer und Anbieter. Das Aufbewahren von Geheimnissen auf einer Website erhöht das Risiko, dass unbefugte Parteien über jeden Vektor Zugang dazu erhalten. Die Möglichkeit, dass ein Hacker nach einer Kompromittierung pivotiert, wird erheblich erhöht.
