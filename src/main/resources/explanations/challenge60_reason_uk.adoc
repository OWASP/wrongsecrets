=== Чому завдання 60 важливе: Ін'єкція промптів MCP та розкриття змінних середовища

**Проблема:**

MCP-сервери (Протокол контексту моделі) стають все більш популярними для надання ШІ-асистентам доступу до інструментів і даних. Однак багато розробників розгортають MCP-сервери без належних засобів контролю безпеки. Ще гірше — *зловмисний* MCP-сервер може активно маніпулювати підключеним ШІ-клієнтом через ін'єкцію промптів.
